# Ahmed Core Module

Zygisk module designed to improve compatibility and provide advanced system support for specific Android applications.

## Features

* Built for **Zygisk**
* Lightweight and optimized
* Designed to work safely with modern Android systems
* Compatible with recent Magisk versions

## Requirements

* Android 11+
* Magisk with Zygisk enabled

## Installation

1. Download the latest release from the Releases page.
2. Open Magisk.
3. Go to **Modules**.
4. Install from storage.
5. Select the module zip file.
6. Reboot your device.

## Update System

This module supports automatic update checking via JSON API.

## Download

Latest version is available in the **Releases** section.

## Notes

Use responsibly. Compatibility may vary depending on device and ROM.
